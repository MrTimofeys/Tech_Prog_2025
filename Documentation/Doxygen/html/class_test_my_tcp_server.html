Подробное описание

Класс для модульного тестирования функционала сервера MyTcpServer.

Класс наследуется от QObject и использует возможности Qt Test Framework для проверки корректности работы методов сервера, включая регистрацию, аутентификацию, генерацию кодов и решение уравнений.
