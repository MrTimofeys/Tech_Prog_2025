double x0,
double tolerance,
int maxIterations )
static

Решает уравнение итерационным методом

Аргументы
funcNameИмя функции для использования ("phi_exp", "phi_quadratic" или "phi_cos")
x0Начальное приближение
toleranceДопустимая погрешность
maxIterationsМаксимальное количество итераций
Возвращает
Приближенное решение или NaN, если нет сходимости или неверное имя функции
Заметки
Метод использует итерации неподвижной точки x_{n+1} = ?(x_n)
Предупреждения
Может не сходиться для некоторых функций и начальных приближений
